# 更新日志

2021.5.29
修复添加第三方网盘下载链接的时自动解析文本的功能

2021.5.21
新增支持添加第三方网盘下载链接的时自动解析文本

2021.5.20
新增支持添加第三方网盘下载链接
新增支持PHP8.0
新增支持匿名（不登录）下载

2021.2.6
新增激活附件管理插件的同时自动安装子插件，目前只支持B2主题的子插件
修复选择自建存储，开启压缩但没有开启压缩密码的时候还是有密码
密码是`undefined`，这是由于前端js没写好判断的问题

2021.2.5
修复分类太多导致分类选择框出现子分类位置错乱的问题
普通的分类选择框改为可以搜索的选择框

2020.10.26
云存储全面强制https
修复批量删除没有关联文章的附件提示 已选附件有关联文章，不允许删除

2020.9.8
更新使用自建存储上传附件时自动选择第一项

2020.9.7
新增memcached支持，可以选择安装redis或者memcached
两者都安装的情况下，优先选择redis
迁移授权验证地址和版本更新地址

2020.8.30
删除本地存储和远程服务功能
增加自建存储功能
把第三方主题适配的代码单独抽离出来做成独立子插件

2020.8.25
新增在线更新
修复云存储上传附件的页面没有显示压缩密码框的bug
修复使用远程存储时在编辑文章时候附件下载地址没有加远程参数的bug
         
2020.8.23
新增本地存储和远程调用服务

2020.8.17
适配RiPro主题(需要修改主题代码，详情查看：安装使用->其他主题)
新增附件关联文章ID，不允许直接删除有关联文章ID的附件
新增插件最新版本检测
新增自定义错误提示页面

2020.8.11 
增加阿里云OSS、又拍云、腾讯云COS前端上传
增加文章与附件关联的记录，不允许直接删除有文章关联的附件
修复redis无设置密码时候验证的错误
修复七牛云前端上传http与https不兼容的bug

2020.8.5 
适配B2主题，增加附件添加的资源属性自定义模板

2020.8.4 
适配腾讯云COS，支持七牛云前端上传
