# 附件迁移

提示

以阿里云和七牛云互相迁移为例，其他的云存储请自行搜索文档进行迁移。

其实最主要是把文件迁移过去，不管你用什么方法，自己上传的也好，用云存储厂商提供的迁移服务也好，只要保证文件路径一致就OK了。

# 从阿里云OSS迁移到七牛云

  1. 第一步先迁移文件

文档地址：如何将阿里云 OSS上 的文件迁移到七牛云存储？

  1. 第二步修改数据库数据
# 把 `dev_fuzqing_files` 其中的`dev_` 数据表前缀改成你自己的，比如默认的`wp_`
UPDATE `dev_fuzqing_files` SET `service_provider`='qiniuyun' WHERE `service_provider`='aliyun';

# 从七牛云迁移到阿里云OSS

  1. 第一步先迁移文件

文档地址：七牛云迁移教程

  1. 第二步修改数据库数据
# 把 `dev_fuzqing_files` 其中的`dev_` 数据表前缀改成你自己的，比如默认的`wp_`
UPDATE `dev_fuzqing_files` SET `service_provider`='aliyun' WHERE `service_provider`='qiniuyun';

# 从自建存储迁移阿里云OSS

  1. 第一步先打包自建存储protected-files文件夹里的所有文件，然后到阿里云OSS后台上传，再解压，注意文件路径一致性。

  2. 第二步修改数据库数据

# 把`dev_fuzqing_files` 其中的`dev_` 数据表前缀改成你自己的，比如默认的`wp_`
# 把`my_self_built_storage1`改成你自己的自建存储别名，如果有多个自建存储，可以添加`sql`语句的`WHERE`条件
# 只有一个自建存储服务的情况下，执行下面的语句
UPDATE `dev_fuzqing_files` SET `service_provider`='aliyun' WHERE `service_provider`='my_self_built_storage1';
# 有多个自建存储服务的情况下，根据实际情况修改
UPDATE `dev_fuzqing_files` SET `service_provider`='aliyun' WHERE `service_provider`='my_self_built_storage1' OR `service_provider`='my_self_built_storage2';
