# 插件升级

不要删除数据

  1. 打开插件的设置页面，务必确认插件卸载的时候不删除插件数据

禁用与删除

  1. 打开已安装插件的页面，先对子插件进行禁用操作（因为子插件依赖主插件），再对附件管理插件进行禁用，然后删除插件

重新上传

  1. 然后上传新版插件进行安装，启用，配置
