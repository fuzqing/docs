# 自定义错误页面

注意

有HTML和CSS基础的用户才推荐修改，没基础的不建议修改（虽然改错了也没啥）

错误页面目录：wp-content/plugins/fuzqing-file-manager/pages/download-error

以下页面都是完全独立的PHP页面，可以说和WordPress一点关系也没有，你也可以当成HTML页面看待

  • 次数限制提示：exceed.php

  • 下载错误提示：error.php

  • 没有登录提示：nologin.php

  • 非法操作提示：illegal-operation.php

  • 关小黑屋提示：backroom.php （小黑屋是B2主题才有的功能）
